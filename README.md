# AI Worker Burnout & Attrition Analysis

## Project Overview

This project explores relationships between workplace conditions, AI-related work patterns, burnout levels, and employee attrition risk.

The analysis focuses on identifying behavioral and organizational patterns associated with employee wellbeing in AI-driven work environments.

---

## Objectives

The main goals of this project are:

- explore factors associated with burnout risk
- analyze relationships between workload, AI usage, and employee wellbeing
- investigate patterns related to attrition risk
- identify potential workplace factors connected to healthier employee outcomes

---

## Dataset

Source:
[AI Worker Burnout & Attrition Risk Dataset](https://www.kaggle.com/datasets/nudratabbas/ai-worker-burnout-and-attrition-risk-dataset)

Dataset characteristics:
- ~1500 observations
- 21 variables
- workplace, behavioral and wellbeing-related features

Note:
The dataset appears to contain synthetic or simulated data, which should be considered when interpreting results and conclusions.

## Research Questions

Some key questions explored in this project:

1. Which workplace factors are most associated with burnout risk?

2. How do AI-related work patterns relate to employee wellbeing?

3. Are there identifiable patterns connected to attrition risk?

4. Which variables appear most strongly associated with healthier workplace outcomes?
